# Basi-registration-API
Basic REST API is a Node.js and Express.js backend application that demonstrates RESTful principles and CRUD operations. It manages user data through structured routes and controllers, following MVC architecture and best practices for scalable API development.

- Create user
- Read users
- Update user
- Delete user
- JSON-based REST API

## Tech Stack
- Node.js
- Express
- Postman

## Run Project
npm install
npm run dev

## API Endpoints

POST    /api/users  
GET     /api/users  
GET     /api/users/:id  
PUT     /api/users/:id  
DELETE  /api/users/:id
